Getting Start With CSS : What is CSS, syntax, Inline, internal and external css:

-------------------------------------------------------------------------------------------------------------------------------------

CSS Means Cascading Style Sheet.

They are three types:

  1. In-line CSS. (Write CSS in line to line)
  2. Internal CSS/Embedded CSS.(Write CSS into the head tag of html)
  3. External CSS.(Link the CSS file into html file )
